本项目是使用 `next.js` 部署在 `vercel` 上的简易博客系统

## 运行方式

1. 运行 `pnpm install` 安装依赖。

2. 运行 `pnpm run dev` 启动本地项目



在浏览器打开 [http://localhost:3000](http://localhost:3000) 即可看到效果 

## 项目介绍

本项目采用 `next.js` 的服务端组件化开发。使用本地 `data.json` 文件作为临时存储方案，具备基本的文章阅读和提交功能。
